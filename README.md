# شيفت (Shift) - حاسبة ساعات العمل والراتب

تطبيق "شيفت" هو تطبيق ذكي لحساب ساعات العمل والرواتب، مبني باستخدام Flutter لتقديم تجربة مستخدم عصرية وسلسة على مختلف المنصات. 

## المميزات الرئيسية
- **إعدادات مرنة:** تخصيص أوقات العمل الرسمية، وحساب الراتب (يومي أو بالساعة).
- **قواعد الإضافي:** إعداد قواعد حساب الوقت الإضافي بشكل كامل.
- **العملات المتعددة:** دعم عملات مختلفة مثل الجنيه المصري، الريال السعودي، الدرهم الإماراتي، والدولار الأمريكي.
- **لوحة تحكم ذكية:** عرض ساعات العمل، الرواتب، الإضافي، وإحصائيات شهرية.
- **إضافة الأيام بسهولة:** إدخال أوقات البداية والنهاية ويقوم التطبيق بالحساب التلقائي.
- **سجل العمليات:** تصفح الأيام السابقة وتعديلها بسهولة.
- **إحصائيات وتقارير:** عرض رسوم بيانية وتصدير التقارير كملفات PDF و Excel.

## التقنيات المستخدمة
- **Flutter**
- **Riverpod** (لإدارة الحالة)
- **SQLite** (قاعدة بيانات محلية)
- **fl_chart** (للرسوم البيانية)
- **pdf & excel** (لتوليد التقارير)

## تشغيل التطبيق
1. تأكد من تثبيت Flutter على جهازك.
2. قم بتنفيذ الأمر `flutter pub get` لتحميل الحزم.
3. قم بتشغيل التطبيق باستخدام `flutter run`.
